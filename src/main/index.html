Mouse & Key Logger

How to use

「右矢印」で操作録画開始。もう一度押すと終了し、シーケンスデータがファイルに保存される。
「左矢印」：シーケンスデータをロードしてシーケンスを開始。
「上矢印」：シーケンスの正常動作確認用の部分スクリーンショットを撮影
「下矢印」：Automan終了。
「右クリック」：keyboard input
「Alt」：情報スクリーンショットの撮影のための短形の開始、終了。一度押すと開始し、もう一度押すと終了してスクリーンショット。
